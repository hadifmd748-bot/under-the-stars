# Under the Stars

A peaceful animated night scene — a boy sitting on a bench, looking up at a sky full of stars. Click anywhere to make a wish.

## Features

- 360 twinkling stars with colored variants (white, blue, amber, pink, purple)
- 12 large cross-flare bright stars
- Ursa Minor constellation with dashed connecting lines
- Full moon with craters and a pulsing amber halo
- Shooting stars that streak across the sky on a loop
- Mountain silhouettes, pine trees, and a reflective lake
- Boy silhouette on a bench with a lantern
- Fireflies drifting through the foreground
- Click anywhere to leave a glowing wish that floats upward
- Rotating stargazing quotes
- Live clock display

## Getting Started

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

The output will be in the `dist/` folder — ready to deploy to any static host (GitHub Pages, Netlify, Vercel, etc.).

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
